# بوت تخمين الرقم

بوت Discord للعبة تخمين الرقم من 1 إلى 100. لعبة جماعية واحدة مع 10 محاولات مشتركة.

## الأوامر

| الأمر | الوصف |
|-------|--------|
| `-تخمين` | بدء لعبة جديدة (صورة فقط) |
| `-ايقاف` | إيقاف اللعبة الحالية |

## القواعد

- لعبة واحدة نشطة في السيرفر
- الجميع يلعبون معًا
- 10 محاولات مشتركة تنقص مع كل إجابة خاطئة
- مهلة دقيقة واحدة للرد، ثم تُنهى اللعبة تلقائيًا
- التلميحات: `الرقم أصغر • المحاولات المتبقية: 9`

## التشغيل المحلي

```bash
cp .env.example .env
npm install
npm start
```

## النشر على Railway

1. ارفع المشروع إلى GitHub
2. أنشئ مشروعًا جديدًا في [Railway](https://railway.app)
3. اربط المستودع واختر **Deploy**
4. أضف متغير البيئة:

| المتغير | الوصف |
|---------|--------|
| `DISCORD_TOKEN` | توكن البوت |

5. فعّل **Message Content Intent** من Discord Developer Portal

Railway يشغّل البوت عبر `npm start` ويفتح منفذ الصحة تلقائيًا على `PORT`.

## المتطلبات

- Node.js 18+
- Discord Bot Token
